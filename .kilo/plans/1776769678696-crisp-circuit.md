# Plan: Product Reference Management System Enhancements

## Project Overview

The user has an existing product management system (React 19 + Express + MySQL + tRPC). The task is to enhance it with:

- Auto-generated reference codes (CC### format)
- Smart duplicate detection with row highlighting
- Smart autocomplete/autofill
- Category management system
- Export to CSV/Excel with timestamps

## Decisions Made

1. **Categories**: Create separate categories table, migrate existing data
2. **References**: Regenerate ALL products to CC### format
3. **Scope**: Implement ALL features (reference generation, autocomplete, duplicate detection, category management, export, keyboard shortcuts)

## Phase 1: Database Schema Changes

### 1.1 Add Categories Table

```typescript
// drizzle/schema.ts - add categories table
export const categories = mysqlTable("categories", {
  id: int("id").autoincrement().primaryKey(),
  userId: int("userId").references(() => users.id, { onDelete: "cascade" }),
  name: varchar("name", { length: 100 }).notNull(),
  slug: varchar("slug", { length: 10 }).notNull(), // first 2 chars uppercase
  createdAt: timestamp("createdAt").defaultNow().notNull(),
});
```

### 1.2 Modify Products Table

- Add `categoryId` FK pointing to categories table
- Update reference field to 5 chars (CC###)
- Add indexes for search optimization
